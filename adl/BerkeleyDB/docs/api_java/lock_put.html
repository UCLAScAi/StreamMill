DbEnv.lock_put

 

import com.sleepycat.db.*;


public native void lock_put(DbEnv lock)
    throws DbException;

Description

The DbEnv.lock_put method releases lock from the lock table.

The DbEnv.lock_put method throws an exception that encapsulates a non-zero error value on failure.

Errors

The DbEnv.lock_put method may fail and throw an exception encapsulating a non-zero error for the following conditions:

EINVAL
An invalid flag value or parameter was specified.

The DbEnv.lock_put method may fail and throw an exception for errors specified for other Berkeley DB and C library or system methods. If a catastrophic error has occurred, the DbEnv.lock_put method may fail and throw a DbRunRecoveryException, in which case all subsequent Berkeley DB calls will fail in the same way.

DbEnv, DbLock
