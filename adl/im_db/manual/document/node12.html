IM_REL_cursor->c_put 

#include <im_db.h>
int IM_RELcursor->c_put(IM_RELC *cursorp, DBT *key, DBT *data, u_int32_t flags);
Description

The IM_RELcursor->c_put function stores key/data pairs into the database.

The flags parameter must be set to one of the following values:

DB_CURRENT

Overwrite the data of the key/data pair referenced by the cursor with the specified data item. The input parameters are ignored.

Return Value

If the cursor is not yet initialized, the IM_RELcursor->c_put function will return DB_NOTFOUND. Otherwise, the IM_RELcursor->c_put function returns 0 on success.
