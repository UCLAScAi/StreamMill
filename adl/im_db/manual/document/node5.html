IM_REL->open 

#include <im_db.h>
int IM_REL->open(IM_REL *relp, char *name, u_int32_t flags);
Description

The IM_REL->open interface opens the database represented by name for both reading and writing. The argument `name' is for Berkeley DB compatibility.

The flags parameter must be set to 0 currently.

Return Value

The IM_REL->open function returns a non-zero error value on failure and 0 on success.
Errors

The IM_REL->open function may fail and return a non-zero error for errors specified for other IM_DB and C library or system functions.
