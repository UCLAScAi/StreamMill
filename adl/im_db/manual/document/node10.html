IM_REL->cursor 

#include <im_db.h>
int IM_REL->cursor(IM_REL *relp, IM_RELC **cursorp, u_int32_t flags);
Description

The IM_REL->cursor function creates a cursor and copies a pointer to it into the memory referenced by ``cursorp''.

A cursor is a structure used to provide sequential access through a database. Currently the flags value must be set to 0.

Return Value

The IM_REL->cursor function returns a non-zero error value on failure and 0 on success.
