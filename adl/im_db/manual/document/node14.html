IM_REL_cursor->c_close 

#include <im_db.h>
int IM_RELcursor->c_close(IM_RELC *cursorp);
Description

The IM_RELcursor->c_close function discards the cursor.

Once IM_RELcursor->c_close has been called, regardless of its return, the cursor handle may not be used again.

Return Value

The IM_RELcursor->c_close function returns a non-zero error value on failure and 0 on success.
