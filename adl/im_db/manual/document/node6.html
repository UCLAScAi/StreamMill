IM_REL->close 

#include <im_db.h>
int IM_REL->close(IM_REL *rel, u_int32_t flags);
Description

The IM_REL->close function closes any open cursors, frees any allocated resources.

The flags parameter must be set to 0 currently.

Once IM_REL->close has been called, regardless of its return, the IM_REL handle may not be accessed again until the handle is re-opened.

Return Value

The IM_REL->close function returns a non-zero error value on failure and 0 on success.
Errors

The IM_REL->close function may fail and return a non-zero error for errors specified for other IM_DB and C library or system functions.
